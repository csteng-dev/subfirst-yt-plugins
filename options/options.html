0

此名單內的影片將**置頂插入首頁最上方、側邊欄**，並在發布新影片時**發送桌面通知**。

此名單內的影片將整合在彈出視窗（Popup）及次要影片清單中，**不會觸發桌面通知與首頁置頂**。

目前尚無次要關注頻道，請於上方新增。

系統運作設定

自訂擴充功能的背景同步頻率、通知提醒與 YouTube 畫面注入設定。

背景同步間隔

設定背景 Service Worker 輪詢各頻道 RSS 訂閱源的頻率。

推播與提示

啟用桌面推播通知

當「⭐ 核心關注 (Tier 1)」頻道上架新影片時，在 Windows 右下角發出系統提醒。

防線展示策略設定

影片時間跨度限制

只展示最近 N 天內發布的影片，過期影片將自動從置頂區塊隱藏。

天內

單一頻道影片上限

每個訂閱頻道在防線中預設最多展示的最新影片數量，防止活躍頻道洗版。

部影片

YouTube UI 注入設定

啟用 YouTube 首頁「置頂防線專區」

在 YouTube 首頁橫幅下方、原生影片網格上方，強行注入一欄以 Bento Grid 排版的「核心關注」新片，徹底繞過首頁演算法推薦。

啟用 YouTube 影片播放頁「側邊欄置頂」

當您在播放任何影片時，將核心頻道的最新精選影片插入到右側推薦清單的最頂部，阻絕演算法的誘惑。

設定備份與數據還原

備份您的關注頻道列表與系統配置，或在更換電腦、重新安裝時進行還原。

匯出備份資料

將目前設定的核心頻道、次要頻道與運作參數匯出為 JSON 檔案，以供日後還原或移轉。

匯入還原資料

選擇先前匯出的 JSON 設定檔，將其匯入本機中。這將會**覆蓋目前**的所有關注清單與設定！

危險區域

此動作將會清除所有快取的影片記錄與關注頻道，重設為初始安裝狀態。
